# Contributing to the FedRAMP 20x GRC-as-Code Engine

We welcome contributions to the `KSI_Engine`. This document outlines the basic guidelines for contributing to this project.

## How to Contribute

1.  **Fork the repository:** Start by forking the repository to your own GitHub account.
2.  **Create a new branch:** Create a new branch for your changes.
3.  **Make your changes:** Make your changes to the codebase.
4.  **Submit a pull request:** When you're ready, submit a pull request with a clear description of your changes.

## Reporting Bugs

If you find a bug, please open an issue on the GitHub repository. Be sure to include a clear and concise description of the bug, as well as steps to reproduce it.

## Suggesting Enhancements

If you have an idea for an enhancement, please open an issue on the GitHub repository. Be sure to include a clear and concise description of the enhancement, as well as the problem it solves.
